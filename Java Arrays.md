При создании массивов (и строк) в памяти выделяются два блока памяти: один для хранения массива (контейнера), второй — под переменную, которая хранит его адрес.

int[] b = a; // В переменную b сохраняем адрес переменной a. a и b указывают на тот же объект-массив в памяти.

объект String в памяти: текст строки в отдельном блоке памяти. в переменной String адрес (ссылка) на объект с текстом.

Слева переменную-массив типа String[] (адрес объекта-массива).
В ячейках объекта-массива String не строки (тексты), а их адреса (ссылки). в переменных String адреса строк (текста).

for (int grade : grades) {
    grade = grade + 1; // не изменит массив, grade — копия значения из массива, а не ссылка на элемент.
}

«переменная-массив хранит ссылку на контейнер, который хранит элементы массива».
переменная-двумерный-массив хранит ссылку на контейнер, который хранит ссылки на одномерные массивы.

Слева «переменная-двумерный-массив», которая хранит ссылку на «объект-двумерный массив». в ячейках ссылки на одномерные массивы — строки двумерного массива.
«контейнер контейнеров» хранит ссылки на «массивы-строки».

int[][] matrix = {
  {1, 2, 3, 4, 5},
  {5, 4, 3, 2, 1}
};

int[] tmp = matrix[0]; // {1, 2, 3, 4, 5}
matrix[0] = matrix[1]; // {5, 4, 3, 2, 1}
matrix[1] = tmp;

for (int row = 0; row < matrix.length; row++){
    for (int col = 0; col < matrix[row].length; col++){
        System.out.print(matrix[row][col] + "\t");
    }
}
