### Приоритеты операторов:
арифметические операторы выше логических.

()
!
==, !=
<, >, <=, >=
&&
||

### Направление операторов
ассоциативность - в каком направлении вычисляются операторы, если стоят «рядом» и имеют один приоритет.

boolean a = true, b = false, c = true;
boolean result = a && b && c;

ассоциативность у && — слева направо. сначала a && b, потом результат с c.

&&, || слева направо
! справа налево

### Выражения (Expression) vs Команды (Statement)
Команду - выполняется, выражение — вычисляется.
У выражения результат вычисления. У результата тип, результат можно присвоить, использовать в другом выражении.

int x; Команда
(a < 10) Выражение, тип boolean
i++; Выражение, тип совпадает с типом переменной i
x = 5; Выражение, тип совпадает с типом переменной x

int x, y, z;
x = y = z = 1; 	// x = (y = (z = 1));

console.nextLine();	// игнорируем результат ввода
Игнорировать результат вычисления выражения, если выражение кроме результата делает еще что-то полезное, и это действие важно, а сам результат — нет.
