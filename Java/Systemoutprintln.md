“Для вывода текста на консоль мы обращаемся к статическому полю out класса System, у которого вызываем метод println(), и в качестве аргумента передаем объект класса String”.

System — классо, роль “моста”, соединяющего вашу программу со средой, в которой она запущена.

out — сущность, хранится внутри System. По умолчанию ссылается на поток вывода на консоль. Подробнее о потоках ввода/вывода в Java можно прочитать здесь.

println — метод, вызывается у сущности out, чтобы обозначить способ, с помощью которого информация будет выведена на консоль.

System — класс, предоставляет разработчику возможности для общения с окружающей его средой: ОС, под которой запущена программа. консоль — приложение, установленное внутри операционной системы (командная строка, Shell для Windows и Terminal для Linux), становится понятно, для чего используется сущность System — для установки связи между нашей программой и “внешним миром”.

Кроме соединения с консолью, у System есть и другой функционал:
Доступ к переменным окружения ОС:

System.getenv("JAVA_HOME")
Возвращает значение переменной окружения JAVA_HOME, устанавливается в системных настройках ОС. При установке Java сталкивался;

Немедленная остановка программы:
System.exit(0)
Прерывает выполнение программы путем остановки Java Virtual Machine;

Получение разделителя строк, который используется в этой ОС:
System.lineSeparator()
Получение текущего времени системы в миллисекундах:

System.currentTimeMillis();

Данные примеры — методы, выполняют определенные действия. останавливают работу программы или возвращают запрашиваемое значение.
Кроме методов, класс System содержит поля, которые хранят ссылки и на другие сущности:
out — ссылка на сущность потока вывода информации на консоль;
in — ссылка на сущность, отвечает за чтение вводимой информации с консоли.
err — похожа out, но предназначена для вывода ошибок.
сущностях внутри класса System. для обращения к элементу, внутри другого элемента, оператор “.”. доступ к сущности потока вывода информации на консоль

out — имя переменной, хранит ссылку на объект (сущность) типа PrintStream. объект настроен, чтобы вся информация, которая записывается в него, попадала на консоль. объект out экземпляром класса PrintStream, и у него можно вызвать соответствующие методы:
printf() — форматированный вывод. Форматирует переданный текст, используя специальные строки и аргументы;

Для вызова метода у объекта оператор “.”. вызов метода println() у сущности out:

println() — метод, вызвать у сущности out.
 
println() — блок команд, направляет текст в поток вывода и в конце добавляет перевод строки.

Чтобы вывести текст на консоль, Java-программисту необходимо сделать следующее:
Обратиться к сущности, которая способна соединить наше приложение и консоль — System;
Обратиться к потоку вывода на консоль — System.out;
Вызвать метод, который записывает информацию на консоль — System.out.println;
Передать текст, который необходимо записать — System.out.println(“Hello World!”);

вывод информации на консоль в Java запускает целую цепочку обращения к различным объектам и методам.
