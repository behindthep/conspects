### Приоритеты операторов:

()	самый высокий
!	высокий
==, !=	средний
<, >, <=, >=	средний
&&	ниже
||	ещё ниже

(арифметические операторы выше логических по приоритету.)

### Направление операторов
ассоциативность - в каком направлении вычисляются операторы, если стоят «рядом» и имеют один приоритет.

boolean a = true, b = false, c = true;
boolean result = a && b && c;

ассоциативность у && — слева направо. сначала a && b, потом результат с c.

&&, || слева направо
! справа налево

### Выражения vs Команды
команды (Statement) и выражения (Expression). Про команду говорят, что она выполняется, а про выражение — что оно вычисляется.
у выражения есть результат вычисления. У результата,есть тип, результат можно присвоить, использовать в другом выражении.

int x; Команда
(a < 10) Выражение, тип boolean
i++; Выражение, тип совпадает с типом переменной i
x = 5; Выражение, тип совпадает с типом переменной x

int x, y, z;
x = y = z = 1; 	// x = (y = (z = 1));

console.nextLine();	// игнорируем результат ввода
Игнорировать результат вычисления выражения нужно, например, если выражение кроме результата делает еще что-то полезное, и нам это действие важно, а сам результат — нет.

тернарный оператор является выражением, его результат можно присвоить.
