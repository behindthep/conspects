### Базисы
- Не конспектируй, то что легко гуглится. Держи конспекты в читаемом виде.
- К проектам свои идеи. Пиши код последовательно: сделал шаг - отладил - след. шаг.
- Понимай что делаешь и зачем. Как конструкции решают реальные задачи. Обучение - поток, темы добавляются когда нужно и только, чтобы решить очередную задачу.
- Читай чужой код, код библиотек, которые используешь на GitHub.
- Программист - тот, кто решает задачи бизнеса с помощью кода. Критерии готовности для работы: способность выполнить тестовые задания; написать работающий проект с нуля (таск-менеджер, API-сервис), а не иметь множество отрывочных знаний.
- Отладчик, программу встроенную в редактор, выполнять код по шагам.

### Продвинутый гуглинг
- site:stackoverflow.com how to test react code — поиск среди страниц сайта Stackoverflow.
- add class to element -jquery — дефисом стоп-слова, найдены все варианты, в которых эти слова не упоминаются.
- "immutable js" — двойные кавычки искать точное совпадение.
- github js http client — найти библиотеку для выполнения HTTP-запросов в JavaScript.

### Резюме
Инструмент продажи навыков - чётким, ориентированным на пользу работодателя и подкреплённым примерами. Умещаться на 1-2 стр.
Релевантность вакансии. Для Frontend - упор на JS-фреймворки, для DevOps - на CI/CD и облака.
Разделы:
  - Контакты: почта, телефон, гитхаб, линкедин, портфолио.
  - О себе (2-3 строки): кто вы, ключевые технологии, чего хотите. "Backend-разработчик (Java, SpringBoot). Ищу продуктовую команду."
    Конкритка вместо общих фраз:
    BAD "Ищу работу начинающим программистом"
    NICE "Разработал 3 веб приложения на React, включая интернет-магазин с 100+ пользователями."
  - Навыки: Сгруппируйте по темам: backend, frontend, бд, devops...
  - Проекты: название, цель, стек, мой вклад. "REST API для бронирования (Python, FastAPI). Ускорил запросы на 20%."
  - Опыт работы: компания, роль, даты. Удаляй нерелевантый опыт. Упор на достижения.
    - "Улучшил производительность API, сокртаив время обработки запроса с 200 мс до 50 мс"
    - "Добавил тесты, увеличив покрытие кода с 60% до 90%".
  - релевантное Образование: ВУЗ, курсы, сертификаты.

Проверить через LanguageTool или попроси ИИ прочитать на Ошибки и опечатки.
Избегать Шаблонные формулировки: "ответственны, целеустремлённый" - не информативно.
